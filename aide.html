Points de terminaison

Obtenir tous les messages

GET http://127.0.0.1:8000/api/Messages

Ajouter un message

POST http://127.0.0.1:8000/api/Messages
Content-Type: application/json

{
"titre": "Le titre de votre message",
"description": "La description de votre message",
"auteur": "Votre e-mail",
"langue": "Langue du message"
}

Supprimer un message

DELETE http://127.0.0.1:8000/api/Messages/:id
